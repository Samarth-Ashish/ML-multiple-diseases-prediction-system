# 🏥 MediScan AI - Disease Prediction System

A sophisticated **Streamlit web application** for disease prediction using ensemble machine learning models. Features a beautiful dark-themed medical interface inspired by modern health-tech applications.

![Streamlit](https://img.shields.io/badge/Streamlit-1.31+-red)

## 🌟 Features

- **5 Disease Prediction Models:**
  - 🩸 **Diabetes** (Pima Indians Dataset)
  - 🔬 **Anemia** (Complete Blood Count Analysis)
  - 💜 **Kidney Disease** (Chronic Kidney Disease)
  - 🫀 **Liver Disease** (Liver Patient Dataset)
  - 🦋 **Hypothyroid** (Thyroid Function Tests)

- **Ensemble ML Models:** Random Forest + XGBoost + LightGBM
- **Advanced Preprocessing:** Imputation, Scaling, SMOTE balancing
- **Beautiful UI:** Dark theme with gradient accents, color-coded by disease
- **Real-time Predictions:** Instant risk assessment with confidence scores
- **Interactive Interface:** Easy-to-use form inputs with validation ranges

## 📋 Prerequisites

Before running the app, you need:

1. **Python 3.8 or higher**
2. **Required datasets** (CSV files):
   - `anemia.csv`
   - `diabetes.csv`
   - `hypothyroid.csv`
   - `kidney_disease.csv`

3. **Trained model files** (generated by running the Jupyter notebook)

## 🚀 Installation

### Step 1: Clone or Download

Download all project files to a single directory.

### Step 2: Install Dependencies

```bash
pip install -r requirements.txt
```

Or install individually:

```bash
pip install streamlit pandas numpy scikit-learn xgboost lightgbm imbalanced-learn
```

### Step 3: Train the Models (REQUIRED FIRST TIME)

**Important:** You must run the Jupyter notebook to train and save the models before using the Streamlit app.

1. Open `UPDATED_Disease_PREDICTION.ipynb` in Jupyter Notebook or Google Colab
2. Upload the required CSV datasets when prompted
3. Run all cells in order
4. This will generate 25 `.pkl` files (5 files per disease):
   - `{disease}_model.pkl`
   - `{disease}_scaler.pkl`
   - `{disease}_imputer.pkl`
   - `{disease}_feature_names.pkl`
   - `{disease}_threshold.pkl`

### Step 4: Run the Streamlit App

```bash
streamlit run disease_prediction_app.py
```

The app will open in your browser at `http://localhost:8501`

## 📊 Usage Guide

### 1. Select a Disease

Click on one of the 5 disease cards at the top:
- 🩸 Diabetes
- 🔬 Anemia

### 2. Enter Patient Data

Fill in the laboratory values for the selected disease. Each field includes:
- **Input validation** (min/max ranges)

### 3. Run Prediction

Click the **"⚡ Run AI Prediction"** button to:
- Load the trained ML model

The app displays:
- **Prediction:** POSITIVE or NEGATIVE

- **Recommendations:** Next steps based on results

## 🧠 Model Architecture

### Training Pipeline

1. **Data Loading & Cleaning**
   - Duplicate removal
   - Missing value handling
   - Zero replacement for invalid medical values

2. **Preprocessing**
   - SimpleImputer (median strategy)
   - StandardScaler normalization
   - SMOTE oversampling for class balance

3. **Model Training**
   - Ensemble voting classifier
   - RandomForest, XGBoost, LightGBM
   - Recall-optimized for medical screening

4. **Threshold Optimization**
   - Custom thresholds for each disease
   - Maximizes recall (sensitivity)
   - Minimizes false negatives

### Input Features

#### Diabetes (8 features)
- Pregnancies, Glucose, Blood Pressure, Skin Thickness
- Insulin, BMI, Diabetes Pedigree Function, Age

#### Anemia (4 features)
- Hemoglobin, MCH, MCHC, MCV

#### Kidney Disease (14 features)
- Age, BP, Specific Gravity, Albumin, Sugar, Blood Glucose
- Blood Urea, Serum Creatinine, Sodium, Potassium, Hemoglobin
- RBC, Pus Cells, Bacteria

#### Liver Disease (10 features)
- Age, Gender, Total Bilirubin, Direct Bilirubin
- Alkaline Phosphatase, ALT, AST, Total Proteins
- Albumin, A/G Ratio

#### Hypothyroid (7 features)
- Age, Sex, TSH, T3, TT4, T4U, FTI

## 🔧 Troubleshooting

### "Model files not found" Error

**Solution:** Run the Jupyter notebook first to train and save models.

```bash
# Check if model files exist
ls -la *.pkl
```

You should see 25 `.pkl` files (5 per disease).

### Import Errors

**Solution:** Install all dependencies:

```bash
pip install -r requirements.txt --upgrade
```

### Data Loading Issues

**Solution:** Ensure CSV files are in the same directory as the notebook and have correct filenames.

### Port Already in Use

**Solution:** Use a different port:

```bash
streamlit run disease_prediction_app.py --server.port 8502
```

## 🔄 Updating Models

To retrain models with new data:

1. Update the CSV dataset files
2. Re-run all cells in the Jupyter notebook
3. New `.pkl` files will overwrite old ones
4. Restart the Streamlit app

## 🎯 Performance Metrics

The models are optimized for **high recall** (sensitivity) to minimize false negatives in medical screening:

- **Recall Priority:** Catches most positive cases
- **F1-Score Balanced:** Overall performance
- **Precision Considered:** Reduces false alarms
- **Custom Thresholds:** Per-disease optimization
